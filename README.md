# 🧮 CodeAlpha Calculator

A modern and responsive calculator developed using **HTML, CSS, and JavaScript** as part of the **CodeAlpha Frontend Development Internship**. The calculator performs basic arithmetic operations with a clean, iOS-inspired user interface and supports both mouse and keyboard input.

## 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 📊 Percentage Calculation
* 🗑️ Clear (AC) Function
* ⌫ Delete Last Character
* ⌨️ Keyboard Support
* 📱 Fully Responsive Design
* 🎨 Modern iOS-inspired User Interface
* ⚡ Smooth Hover and Click Animations

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📂 Project Structure

```text
CodeAlpha_Calculator/

```

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click **index.html** or open it with **Live Server** in Visual Studio Code.
4. Start performing calculations.
